# week1andweek2assignment
Overview

This project contains 10 Java implementations of common hash table, caching, and system design problems, suitable for interview preparation and real-world applications. Each problem is implemented as a standalone Java class, runnable in IntelliJ IDEA.

The project covers:

Social Media Username Checker

E-commerce Flash Sale Inventory Manager

DNS Cache with TTL

Plagiarism Detection System

Real-Time Analytics Dashboard

Distributed Rate Limiter

Autocomplete System for Search Engine

Parking Lot Management with Open Addressing

Two-Sum Variants for Financial Transactions

Multi-Level Cache System (L1/L2/L3)

✅ Features

Hash Table Implementations: Key-value lookups with O(1) performance

Concurrency & Thread Safety: Inventory manager & rate limiter support safe concurrent operations

Real-Time Systems: Analytics dashboard & rate limiter handle live updates efficiently

Caching & LRU: DNS cache, multi-level cache, and autocomplete use LRU eviction

Open Addressing: Parking lot management handles collisions with linear probing

Frequency Tracking: Autocomplete, username checker, and analytics dashboard maintain popularity statistics

Right-click on the file → Run 'ProblemX.main()'

Check the console output for sample results
